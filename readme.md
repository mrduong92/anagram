## Install

```
$ docker-compose build
// Run first only

$ docker-compose up -d

```

## SSH to homestead
```
$ ssh homestead@127.0.0.1
// Password: secret
```

## Mysql
```
$ docker-compose exec mariadb /bin/bash
$ mysql -uadmin -psecret;
```
